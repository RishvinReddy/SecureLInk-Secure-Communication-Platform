## Overview
SecureLInk is a secure communication platform implementing hybrid encryption
(AES + RSA) with authentication and message protection.

## Tech Stack
- Frontend: React + TypeScript
- Backend: Flask (Python)
- Encryption: AES, RSA
- Auth: JWT
- Database: SQLite

## Features
- User authentication
- Encrypted message exchange
- Secure key handling
- Modular backend architecture

## Setup Instructions

### Backend
1. Create virtual environment
2. Install dependencies:
   \`pip install -r requirements.txt\`
3. Create \`.env\` using \`.env.example\`
4. Run:
   \`python app.py\`

### Frontend
1. Install dependencies:
   \`npm install\`
2. Start:
   \`npm start\`

## Security Note
Environment variables and secrets are excluded from version control
for security reasons.
